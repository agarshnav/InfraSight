# InfraSight Architecture

## Frontend Layer
- Next.js dashboard
- Realtime updates via WebSockets

## Backend Layer
- FastAPI backend
- JWT authentication
- Metrics ingestion APIs
- Alerting services

## Data Layer
- PostgreSQL for relational data
- Redis for caching and realtime communication

## Observability Layer
- Prometheus metrics
- Grafana dashboards
- Loki log aggregation

## AI Layer
- anomaly detection
- incident summarization
- future RAG integration

## Infrastructure Layer
- Dockerized local development
- Kubernetes planned for future deployment
